Microsoft Excel - Making your work easier

BY BLACK BOARD LEARNING
“The global economy is built on two things: the internal combustion engine and Microsoft Excel. Never forget this.” — Kevin Hector, Twitter

Microsoft Excel is an amazing software to manage a huge amount of data and hence it is being widely used in every industry. Microsoft Excel is nothing but a spreadsheet program that is used to record and analyse numerical data. Excel, as all spreadsheet applications, uses a collection of cells arranged into rows and columns to organize and manipulate data. You can use Excel to enter all sorts of data and perform mathematical, financial, or statistical calculations. They can also display data as charts, histograms, and line graphs. Hence, Excel is considered as a powerful data visualization and analysis tool.

Excel is a powerful and valuable tool for many businesses, which use it to plan budgets, chart data, record income and expenditures, and other fiscal plans. Excel has become a standard in the business world since it helps in maintaining all the data in one location. It provides major help for a businessman in developing strategies. Not only for businesses, Excel can also be used for personal purposes like managing the expenses, keeping track of loans and planning for investments.

Basic features of Excel - Workbooks, Worksheets and Cells

An excel file in general, is called as a workbook. A workbook contains one or more spreadsheets, called as worksheets to help you organize your data.

To create a new workbook:
Open Excel -> Select a Blank workbook (or press Ctrl+N) -> Start typing

Worksheet consists of rows and columns that form a table. Alphabetical letters are usually assigned to columns and numbers are usually assigned to rows. The point at which a column and a row meet is called as a cell. The address of a cell is given by the letter representing the column and the number representing a row.

Basic functions you can perform in Excel
Format Cells

You can use Format cells to change the number, alignment, font style, Border style, Fill options and Protection. You can find this option with right click of the mouse. After right-clicking, pop-up will appear, and then you need to click on Format Cells.

Merge & Center

To merge 2 or more cells, you can select the required cells and then select Merge & Center option under the Home tab

Find & Select

If you want to find a specific value in the large amount of data you have, use the following steps:

i. On the Home tab, in the Editing group, click Find & Select.
ii. Click Find. The 'Find and Replace' dialog box appears.
iii. Type the text you want to find.
iv. Click 'Find Next'. The first occurrence of the searched value will be highlighted.
v. Click 'Find Next' to select the second occurrence.
vi. To get a list of all the occurrences, you can click 'Find All'.

Hide Columns or Rows

First, select the columns or rows you want to hide. Right-click the selected column or row you want to hide and then choose Hide.
You can also click the Home tab and in the Cells group, Click Format -> Hide & Unhide and pick either Hide Rows or Hide Columns, as required.

AutoFill

You can use AutoFill in Excel to automatically fill a series of cells.

Excel Template

A template is a predesigned excel workbook which you can use to create new workbooks with the same formatting and predefined formulas. You can create a new excel spreadsheet, if you want you can update the required headers and formula which you might be be using repeatedly, and then save the file as Excel Template(*.xltx).

Later when you open a new workbook, you can select this template from the File Tab instead of selecting a Blank workbook. Also, many excel templates are available online (provided by Microsoft Office) for budgets, inventory, schedules, Gantt charts, timelines, and more.

Working with formulas in Excel

A formula is an expression which calculates the value of a cell. Functions are predefined formulas and are already available in Excel.

To enter a formula, execute the following steps.
1. Select a cell.
2. To let Excel know that you want to enter a formula, type an equal sign (=).
3. For example, if you have a value of 5 in A1 and 3 in A2 and if you type the formula =A1+A2 in A3, it will give you a value of 8.

By default, Excel follows an order in which it executes the formula. It can be explained with the BODMAS rule.

BODMAS is an acronym for Brackets Order Division Multiplication Addition Subtraction i.e., If a part of the formula is in parentheses, that part will be calculated first. It then performs multiplication or division calculations. Once this is complete, Excel will add and subtract the remainder of your formula.

To insert a function, execute the following steps.
1. Select a cell.
2. Click the Insert Function button. The 'Insert Function' dialog box appears.
3. Search for a function or select a function from a category. For example, choose COUNTIF from the Statistical category.
4. Click “Ok”.

Features for analysing data in Excel
Sort

You can always arrange your data if needed, before doing any analysis in Excel.
Click on any cell in your data set. Then select Sort in the Sort & Filter group of the Data tab. You can choose the column by which you are sorting, sort on value and the order by which you want to sort.

Filter

Excel provides you a feature to filter the data based on a specific value or condition.
Click on any cell in the data set. Then select Filter in the Sort & Filter group of the Data tab.
Arrows will appear near the column headers. Click on the arrow in the column by which you want to filter. You can select the value or the category or you can give the condition by which you want to filter.

You can always clear the filter, by clicking on the arrow of the column and then selecting Clear filter option. If you want to clear the filter option from all the columns, you can re-click on the Filter option in the Data tab. The filters will be disabled.

Conditional Formatting

It is used to highlight the cells which satisfy a given condition.

i. Select the cells in which you want to apply the condition.
ii. On the Home tab, in the Styles group, click Conditional Formatting.
iii. You can select any condition as needed, from highlight Cell Rules.
iv. You can then select the colour format with which you want to highlight the cells.
v. There are more options available to highlight the cells such as Data Bars, Color Scales and Icon Sets.
vi. You can also create a new condition by using the New rule option under Conditional Formatting. With this you will be able to give any formula as condition also. You can edit or delete the rules with Manage Rules option.

Remove Duplicates

One of the problems with a large amount of data is redundancy. Excel can help you in identifying these unwanted repeated data by removing the duplicate values.
1. Click any single cell inside the data set.
2. On the Data tab, in the Data Tools group, click Remove Duplicates.
3. Select all the columns and click Ok.
4. Excel will remove all the identical rows except for the first one. A pop up will appear showing the number of duplicate records that had been found and removed.

Data Validation

Data validation is used to control what a user can enter in a cell. For example, you could use data validation to make sure a value is a number between 1 and 6, make sure a date occurs in the next 30 days, or make sure a text entry is less than 25 characters.

1. Select the cells for which you want to apply the data validation. For example, the cell A1 is selected.
2. On the Data tab, in the Data Tools group, click Data Validation.
3. In the Settings tab of the Data validation pop-up, you can select the validation type you want to apply in the Allow list. For example, Decimal is selected.
4. Then from the Data list, you can select any condition and provide appropriate inputs.
5. Click Ok. For example, the condition is given as Between 100 and 200. Now, in the selected cells, you will be able to enter a decimal number between 100 and 200 only. If you enter any number apart from this range, you will get an error.

In addition to the above mentioned features, there are several other functionalities in Excel such as What-if analysis, Scenario manager, Pivot tables and charts, as well other data analysis tools to perform tests such as ANOVA, Regression, T-tests, etc.

We all are dealing with numbers in our day-to-day life in one way or the other, ranging from recording one's daily expenses to storing and analyzing complicated data sets. Excel enables us to play with our data as we please. Excel saves our time a lot with its vast features for calculations and visualizations. So it can be said that, with Excel, our lives are made a bit easier.
